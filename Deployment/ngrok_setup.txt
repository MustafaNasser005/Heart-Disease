## üîß Setup Instructions

### Step 1: Install ngrok

#### Windows:
1. Download ngrok from https://ngrok.com/download
2. Extract the zip file
3. Add ngrok to your PATH or place it in your project directory

#### macOS/Linux:
```bash
# Using Homebrew (macOS)
brew install ngrok

# Using Snap (Linux)
sudo snap install ngrok
```

### Step 2: Authenticate ngrok

1. Sign up at https://ngrok.com/
2. Get your authtoken from the dashboard
3. Run authentication:
```bash
ngrok config add-authtoken YOUR_AUTHTOKEN_HERE
```

### Step 3: Start Your Streamlit App

```bash
# Navigate to your project directory
cd "Heart Disease"

# Start the Streamlit app
cd UI
streamlit run app.py
```

Your app should now be running at `http://localhost:8501`

### Step 4: Expose with ngrok

Open a new terminal window and run:

```bash
# Expose the Streamlit app
ngrok http 8501
```

### Step 5: Access Your App

ngrok will provide you with a public URL like:
```
Forwarding    https://abc123.ngrok.io -> http://localhost:8501
```

Share this URL with others to access your app from anywhere!

## üõ†Ô∏è Troubleshooting

### Common Issues:

1. **Port Already in Use**:
   ```bash
   # Kill process using port 8501
   lsof -ti:8501 | xargs kill -9
   ```

2. **ngrok Not Found**:
   ```bash
   # Ensure ngrok is in PATH or use full path
   ./ngrok http 8501
   ```

3. **Authentication Failed**:
   ```bash
   # Re-authenticate ngrok
   ngrok config add-authtoken YOUR_AUTHTOKEN_HERE
   ```

4. **App Not Loading**:
   - Check if Streamlit is running on port 8501
   - Verify all dependencies are installed
   - Check the ngrok tunnel status
